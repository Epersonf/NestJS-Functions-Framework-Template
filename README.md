## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with Cloud Functions debug and deploy system.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start:local
```

## Deploying the app

```bash
$ npm run deploy:dev
$ npm run deploy:hml
$ npm run deploy:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
